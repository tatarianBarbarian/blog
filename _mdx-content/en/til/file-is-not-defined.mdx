---
title: 'ReferenceError: File is not defined in Node.js'
date: '2024-08-07'
excerpt: 'Fixing ReferenceError: File is not defined'
description: 'ReferenceError: File is not defined'
---

`File` class is available in Node.js starting from v20, so updating Node.js should fix the problem.

If by some reason updating is impossible then use [File polyfill for Node](https://www.npmjs.com/package/@web-std/file)
